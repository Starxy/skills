# 论文类型路由索引

读取本索引后选择一个主类型；混合型论文至多再选择一个辅助类型。随后只加载所选镜头文件。

| 分支 | 选择信号 | 镜头文件 |
| --- | --- | --- |
| `实验研究` | 以研究问题或假设、样本、测量、对照和经验结果为论证主轴。 | [experimental.md](experimental.md) |
| `算法 / 系统` | 以计算任务瓶颈、算法机制、系统设计、实现流程或性能权衡为主轴。 | [algorithm-system.md](algorithm-system.md) |
| `理论论文` | 以形式化问题、假设、定理、证明或理论保证为主轴。 | [theory.md](theory.md) |
| `定性研究` | 以访谈、观察、文本或档案材料及其编码和解释为主轴。 | [qualitative.md](qualitative.md) |
| `综述 / 观点` | 以文献综合、概念框架、立场论证、风险分析或研究议程为主轴。 | [review-perspective.md](review-perspective.md) |
