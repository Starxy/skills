# 论文类型镜头

先选择一个主类型；混合型论文至多再选择一个辅助类型。只读取并应用所选镜头，不叠加全部规则。

| 分支 | 选择信号 |
| --- | --- |
| `实验研究` | 以研究问题或假设、样本、测量、对照和经验结果为论证主轴。 |
| `算法 / 系统` | 以计算任务瓶颈、算法机制、系统设计、实现流程或性能权衡为主轴。 |
| `理论论文` | 以形式化问题、假设、定理、证明或理论保证为主轴。 |
| `定性研究` | 以访谈、观察、文本或档案材料及其编码和解释为主轴。 |
| `综述 / 观点` | 以文献综合、概念框架、立场论证、风险分析或研究议程为主轴。 |

## 实验研究

- **核心贡献：** 在明确研究问题下，通过合适设计和测量获得新的经验认识；贡献可以是效应、机制线索、测量方法或反例，而不只是“结果显著”。
- **支持证据：** 对齐假设、样本选择、对照、变量操作、测量可靠性、效应量与不确定性，并检查稳健性、替代解释及设计实际允许的因果强度。
- **避免过度声称：** 不把相关性写成因果，不把单一人群或情境推广为普遍规律，也不把统计显著自动等同于实际重要。

## 算法 / 系统

- **核心贡献：** 针对任务瓶颈提出新的机制、模块组合、训练或运行方式，并形成能力、效率、可扩展性或可靠性上的可归因改进。
- **支持证据：** 对齐强基线、等预算比较、消融、效率与资源成本、鲁棒性、误差分析和跨设置验证，说明每类证据具体支持哪个设计主张。
- **避免过度声称：** 不把更高分数自动等同于算法创新，不把组件共现写成组件有效，也不从有限任务推出对所有任务或规模都成立。

## 理论论文

- **核心贡献：** 在明确问题设定和假设下，给出新的定理、界、可识别性结果、证明策略、反例或对已有理论的统一认识。
- **支持证据：** 对齐定义、假设、定理陈述、证明主线、关键引理、构造或反例，并比较新增保证相对已有结果改变了什么。
- **避免过度声称：** 不把特定假设下的结论推广到假设外，不把渐近或存在性保证写成现实性能，也不把未验证的直觉当成定理结论。

## 定性研究

- **核心贡献：** 从系统收集的材料中提出新的经验主题、概念关系、过程机制或情境化解释，并说明其分析视角带来的认识。
- **支持证据：** 对齐样本与场域选择、资料获取、编码或解释过程、参与者材料、研究者反思、三角验证、负面案例和资料饱和度。
- **避免过度声称：** 不把研究者解释写成参与者原话，不把小样本洞察写成人群比例或普遍因果规律，也不把可迁移性等同于统计代表性。

## 综述 / 观点

- **核心贡献：** 对既有材料形成新的综合、分类、概念框架、中心论点、风险链条或研究议程；贡献在论证组织与认识增量，不必硬套新算法或新实验。
- **支持证据：** 对齐材料选择方式、来源覆盖、概念区分、论证链、案例与反例，并标明哪些结论来自领域共识、作者判断或未来预测。
- **避免过度声称：** 不把作者立场写成领域共识，不把选择性案例写成总体因果证据，也不把规范性建议或预测写成已经验证的事实。
